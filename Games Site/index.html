The Seven Continents Game

Game Made by Divyesh Idhate in Python

This is a game made for educational purposes by me. It is made in Python using turtle library. It consists of a blank world mapand you are opt to enter the names of continents. The game progresses as you enter correct games. You can type "Exit" to end the game and the contitnents which were not inputted by you will be saved in a file "continents_to_learn.csv" so that students can review what they have missed out. The code for this project and be found in my repo: The_Continents_Games. The code was turned into exe and has been compressed in a rar file.
